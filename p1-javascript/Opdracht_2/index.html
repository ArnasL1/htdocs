Wat is het verschil met de script tag boven in te laden in de < head > tag t.o.v onder inladen aan het einde van je HTML tag

Het bepaalt wanneer het script wordt geladen. Script in < head > wordt geladen voor < body > content. Als script onderaan < body > staat, wordt het geladen na alle < body > structuren.

Wat doet de attribuut “defer” in de < script > tag

attribuut "defer" zorgt ervoor dat het < script > parallel aan de pagina wordt gedownload, maar alleen wordt uitgevoerd aan het einde van het laden van de pagina.

Wat is het verschil tussen HTML en JavaScript?

HTML is een opmaaktaal, terwijl Javascript een programmeertaal is. Javascript staat dynamische elementmanipulatie toe, terwijl HTML voor paginastructuur is.
